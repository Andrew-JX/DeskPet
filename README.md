# DeskPet

把你自己的宠物变成陪你上班的桌面伙伴。

DeskPet 是一个可换皮、可对话、可提醒、可扩展的桌面宠物 Agent。用户上传宠物的绿底图片、精灵图或循环视频，就能把它变成一个透明置顶、可拖动、可互动的桌面伙伴。它不是通用卡通桌宠，而是“我自己的那只”。

> 当前版本：v0.13.0  
> 支持 Windows 安装包 / 便携版，仓库内含示例素材。

## 为什么做

很多桌宠项目在拼记忆、RPG、工具调用，但对普通养宠人来说，最强的需求往往不是“它有多聪明”，而是“它是不是我的宠物”。

DeskPet 的切入点：

- **专属形象**：上传绿底图片、精灵图或视频，快速换成自己的宠物。
- **日常陪伴**：点击互动、聊天、番茄钟、久坐提醒，让宠物参与电脑工作流。
- **可信 AI**：AI 可以聊天、记待办、查专注数据，但写操作必须确认，并留下 trace 审计。
- **本地记忆**：支持最近对话记忆和长期记忆 notes，让它逐渐知道宠物名字、主人偏好和作息习惯。
- **情感边界**：纪念模式定位为“带着它形象的陪伴者”，不冒充已故宠物本体，不消费悲伤。

## 当前能力

### 桌宠与素材

- 透明无框置顶桌宠窗口
- 可拖动、跨屏移动
- 绿幕抠图显示图片/视频
- 素材库上传图片、视频、精灵图
- 拖框选区后按列/行切帧
- 多宠物切换
- 自定义互动动作，动作播完自动回待机
- 点击、聊天、番茄钟、久坐提醒均可绑定动作素材

### AI 与记忆

- 支持 OpenAI 兼容接口和 Anthropic
- 可填写 Base URL、API Key、模型名
- 无 Key 时自动退回本地随机回应
- 最近对话记忆，可设置保留轮数
- 长期记忆 notes，可手动查看、编辑、删除
- API Key 只在主进程使用，渲染层拿不到明文
- 写操作确认：新增/完成待办必须用户点确认
- trace 审计：每次工具调用都有记录

### 小工具

- 番茄钟
- 久坐提醒
- 今日专注分钟数
- 番茄钟完成数
- 互动次数
- 待办新增、查看、完成

## 快速开始

### 普通用户

到 [Releases](https://github.com/Andrew-JX/DeskPet/releases) 下载最新 Windows 包：

- `DeskPet-Setup-x.x.x.exe`：安装版
- `DeskPet-x.x.x.exe`：便携版

首次启动会出现占位宠物：

- **单击宠物**：打开互动菜单
- **右键宠物**：打开后台
- **拖动宠物**：移动位置

### 开发运行

```bash
git clone https://github.com/Andrew-JX/DeskPet.git
cd DeskPet
npm install
npm start
```

### 打包

```bash
npm run pack   # 生成 dist/win-unpacked
npm run dist   # 生成 Windows 安装包和便携版
```

发版时推送 `v*` 标签，GitHub Actions 会自动测试并构建 Release。

## 3 分钟验证路径

1. 启动 DeskPet。
2. 右键宠物打开后台。
3. 在「素材库」上传 `samples/sample-dog-greenscreen.png`。
4. 设置列/行，把精灵图切成帧。
5. 添加宠物，给它选择待机素材。
6. 添加互动，并绑定另一帧或视频素材。
7. 回到桌面，单击宠物触发互动。
8. 在「模型与记忆」里填模型接口，测试连续聊天。

也可以打开 `web-demo/` 做纯网页抠图体验。

## 模型接入

后台「模型与记忆」支持两种接口类型：

| 类型 | 用法 |
| --- | --- |
| OpenAI 兼容接口 | 适合 OpenAI、DeepSeek、通义/智谱等兼容 `/chat/completions` 的服务，填写 Base URL、Key、模型名即可 |
| Anthropic | 使用 Anthropic `/messages` 接口，填写 Key 和模型名 |

也可以通过环境变量临时使用：

```powershell
$env:OPENAI_API_KEY = "your-key"
npm start
```

安全边界：

- Key 不会暴露给渲染层。
- 后台只显示“是否已保存 Key”。
- 可一键清除已保存 Key。
- 长期记忆 notes 可直接编辑删除。
- 最近对话记忆可一键清空。

## 可信工具调用

DeskPet 不把工具执行权交给模型，而是用规则路由判断意图。

流程：

```text
用户输入 -> 规则意图路由 -> 读操作直接执行 / 写操作等待确认 -> 模型包装结果 -> trace 审计
```

示例：

- “今天专注了多久” -> 读取本地统计 -> 直接回答
- “有什么待办” -> 读取待办列表 -> 直接回答
- “记一下：买牛奶” -> 生成待确认提案 -> 用户确认后写入

这让桌宠可以变聪明，但不越权。

## 项目结构

```text
main.js        Electron 主进程：窗口、IPC、配置读写、AI 调用、工具调用
intent.js      规则意图路由
preload.js     contextBridge 白名单 API
pet.html       桌宠窗口：透明显示、抠图、拖动、聊天、菜单
panel.html     后台：素材库、宠物配置、模型与记忆、统计和 trace
samples/       示例绿底精灵图
assets/        开发模式运行数据，打包后用户数据写入 userData
tests/         意图路由单测
```

## 已知限制

- 复杂背景素材需要先处理成纯绿底，当前抠图阈值偏轻量。
- `.mov` 播放依赖系统编码支持，推荐使用 `.mp4` 或 `.webm`。
- 真实模型联网对话需要用户自行提供可用 API Key。
- 纪念模式只做陪伴表达，不做“复活式人格模拟”。
