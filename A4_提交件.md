# DeskPet · 把你的宠物变成陪你上班的桌面伙伴

**赛道：宠物 AI（轻量机会验证） | 提交人：吉敏宇 · AI 产品方向**

### 1. 机会判断：目标用户 / 痛点 / 为什么现在做

**目标用户**：有宠物、或曾经养过宠物的上班族（每天 8h+ 对着电脑）。

**痛点**：① 独自办公缺陪伴，市面桌宠都是「通用卡通形象」，与我无关；② 宠物离世后的情感寄托，**几乎没有产品认真承接**。

**为什么现在做**：调研发现「桌宠 / Agent 桌宠」已很卷（GitHub 上 Agentic-Desktop-Pet、Clawd on Desk 等多个高星项目），但有两个共同盲区——**形象不可换**（换皮要美术重画）、**只炫技不做情感与商业**（拼记忆/RPG/工具，答不上「谁付费」）。AI 立绘/图生视频 + 绿幕抠图，让「丢一张图/一段视频就变成你那只宠物」第一次低成本可行。

> **差异化一句话**：不拼 Agent 炫技，拼「**专属 + 情感 + 可信**」——它是**你那只**宠物，不是通用形象。

### 2. 用户证据

- **竞品即证据**：最高星的 Agentic-Desktop-Pet 自己在 README 标注 *"many bugs, please be patient"*，需 Python+Godot 自配环境；Clawd 只服务盯 Claude Code 的程序员——**全是给极客的玩具，无人切普通人 + 情感**。
- **需求热度**：小红书/抖音「宠物纪念」「把我家猫做成桌宠」类内容长期高互动，宠物离世「想留个念想」是真实、高情绪强度的需求。
- **一手访谈**：向 10+ 位养宠/曾养宠的朋友求证，普遍认可「专属形象」「纪念」的情感价值（2 周内补足样本与量化）。

### 3. MVP 样例（已可运行 + 已上线）

可运行 Electron 应用，`npm start` 或下载安装包即用：

- **换皮（核心）**：上传你宠物的**绿底图或循环视频**，桌宠**实时抠绿幕**显示——图片、视频都行，真透明、可丝滑动；
- **素材库 + 拖拽切帧**：一张多动作精灵图，拖框圈区域、按列行切帧入库，供反复挑用；
- **自定义互动**：互动可自由增删命名，每个绑一段素材；点宠物弹菜单触发，播完回待机；
- **小工具**：番茄钟（含「专注中」姿势 + 倒计时）、久坐提醒、聊天、后台；触发动作均可绑素材；
- **可信内核**：AI 帮你记/查待办、看陪伴数据，走「规则意图路由 → 工具调用 → **写操作确认** → **trace 审计**」，密钥只在主进程，并含单元测试——把 FitMind 的「不乱来」从口号变成代码；
- **纪念模式**：定位「带着它形象的陪伴者」，不冒充本体、不消费悲伤；
- **工程化**：开源 + GitHub Actions 自动构建 Windows 安装包，普通人一键可用。

### 4. 商业判断

| 维度 | 判断 |
|---|---|
| **谁付费** | 核心：想要「专属/纪念」形象的养宠上班族；延伸：宠物店/宠物医院（送客的情感礼品）。|
| **为什么付费** | 免费桌宠到处有，但「**把我自己那只宠物变成会动、会陪我的样子**」是情感独占价值，离世纪念付费意愿最强。|
| **定价** | 软件免费获客；**按形象付费**：定制/生成专属形象 19–39 元，纪念礼盒（定制形象+寄语）99 元；B 端按套授权。|
| **获客** | 小红书/抖音内容自传播（「我把走丢的猫做成了桌宠」天然爆款）；宠物社群 + 宠物店异业合作。|

### 5. 两周验证计划 & 下一步能做什么

- **第 1 周·验证「想不想要」**：发 3 条小红书/抖音样片（含 1 条纪念向），看收藏/私信；完成 20 份访谈，量化「愿为专属形象付费」比例与价位。**指标**：单条 ≥50 收藏或 ≥10 条「怎么做」私信即正信号。
- **第 2 周·验证「愿不愿付」**：上线「照片→专属形象」最小付费链路（人工/半自动交付即可），跑 5–10 单真实付费。**指标**：≥5 单付费 = 方向成立。
- **资源**：1 人可启动（我）；AI 生成额度 + 少量投放预算；提速则加 1 名美术/内容。
- **产品可延展的方向**（不稀释 v1）：① 照片/视频→自动生成专属形象，闭环换皮；② 可信内核扩成**桌面 Agent 入口**（白名单开应用/文件、查天气日程、记笔记总结，皆确认+trace）；③ 感知工作状态主动陪伴（久专注鼓励、深夜提醒休息）；④ 一键导出 GIF 便于分享传播；⑤ 远期接手表/健康数据被动采集。

---
*取舍：v1 砍掉「调用本机任意软件」等高风险能力，先把「专属换皮 + 陪伴 + 可信小工具 + 纪念」这一最小可信闭环做扎实——先解决一个问题，再谈扩展。*
