# 更新日志

本文件记录 DeskPet 的迭代历程（面向开发/维护，不进产品界面）。

## v0.7.0
- 精灵图切帧改为**可视化拖拽**：整张图缩放预览，拖动橙色边框圈出宠物所在区域，按「列/行」自动划分，点格子把该区域裁成独立透明图存入素材库。
- 替代了上一版的「边距/间距」数字输入，所见即所裁。

## v0.6.x
- 引入**素材库（library）**：上传整图/视频，或对一张多动作精灵图设网格后切帧入库；素材可复用。
- 宠物「待机」与每个互动改为**从素材库下拉挑选**（既可挑已有，也可先上传到库再挑）。
- 修复：精灵图源设置网格后切帧入口消失（用 frameIndex/derived 区分「源」与「切出的帧」）。
- 移除界面里的「图生视频提示词生成器」。
- 移除「基本设置」里重复的全局宠物名字（宠物名归每只宠物）。

## v0.5.0
- 重构为**「宠物 + 自定义互动」模型**：一个宠物 = 一段待机素材 + 若干用户可增删改名的互动，每个互动各带素材；触发即播，数秒后回待机；多宠物可切换。
- 渲染层按签名去重，避免统计广播打断视频/重置番茄钟。

## v0.4.0
- **视频模式**：上传绿底循环视频，逐帧实时抠绿幕播放（连续、丝滑）。
- 呼吸效果调柔，仅静帧启用。

## v0.3.0
- 动作模型从「状态=帧区间循环」改为「动作=指定帧」：单帧定格、多帧可循环，解决帧切换的跳动感。
- 后台可视化「帧 → 动作」绑定（该版本后被素材库模型取代）。

## v0.2.0
- **可信工具调用内核**：规则意图路由 → 工具调用（看数据/看待办/记待办）；写操作需确认；调用写入审计记录。
- 确定性陪伴数据（专注时长/番茄钟/互动）由主进程计算。
- 意图路由含单元测试（`npm test`）。

## v0.1.0
- 首个可安装版本：透明无框置顶桌宠 + 绿幕抠图精灵图 + 拖动 + 后台素材管理 + AI 对话 + 番茄钟/久坐提醒 + 纪念模式。
- electron-builder 打包 + GitHub Actions 自动发布 Windows 安装包。
