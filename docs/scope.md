# Scope: 把你自己的宠物变成陪你上班的桌面伙伴——丢一张绿底精灵图就换皮，可对话、会提醒，并支持为已离世宠物的纪念模式。

Forced by `ai-pm-dev prd` to make prioritization explicit.

## Must-haves (v1, max 3)

- ①丢图换皮（绿幕抠图+精灵图状态机）
- ②AI对话+番茄钟/久坐提醒
- ③后台素材与人设管理。

## The one thing (proves the idea)

丢一张图就变成"我那只"宠物的换皮能力——这是与所有通用桌宠的根本差异，最能验证想法。

## Explicitly not doing (v1)

v1不做"调用本机任意软件"等高风险工具能力，也不做手表/健康数据接入；先把专属陪伴这一最小可信闭环做扎实，降低安全与演示风险。
